Metropolis Hastings Algorithm

  1. Start with $\theta^{(0)}$, $t=0$
  2. Propose $\theta′$ according to a transition kernel $q(\theta′|\theta^{(t)})$
  3. Calculate acceptance ratio $$\alpha = \textrm{min}\left( 1, \frac{p(D|\theta′)p(\theta′)q(\theta^{(t)}|\theta′)}{p(D|\theta^{(t)})p(\theta^{(t)})q(\theta′|\theta^{(t)})}\right)$$
  4. Accept $\theta’$ with probability $\alpha$
    • If accepted, set $\theta^{(t+1)} = \theta’$
    • If rejected, set $\theta^{(t+1)} = \theta^{(t)}$
    $t = t+1$. Return to step 2.
